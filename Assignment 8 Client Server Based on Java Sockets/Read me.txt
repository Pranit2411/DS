1. Create a folder and paste the MyServer and MyClient files in it.

2. Open command prompt and navigate to the following folder.

3. Enter the following commands
		javac MyServer.java
		java MyServer

4. Now open another terminal and navigate to the same folder
	
5. Type the following commands
		javac MyClient.java
		java MyClient

6. Enter any message and it will get reflected in terminal 1

7. Then you cn go to terminal 1 and send messge to terminal 2
